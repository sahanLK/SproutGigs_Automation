The purpose of this directory is to create a fake web page for testing the <submitter> module.

This directory only contains the files and other stuffs that require for building the fake web page.
This is separated from both testing and production code. Use this directory only for building the web page.

Always run the builder.py to build the web page and the generated file will be saved into this directory.
After generated the html file, move the html file into the active directory of the local web server and
then run the test_submitter.
